# Analog Audio Equalizer (LTspice)

## Overview
This project implements an analog audio equalizer using operational amplifiers.

## Features
- Frequency separation (Bass, Mid, Treble)
- Adjustable gain using potentiometers
- Summing amplifier output

## Tools Used
- LTspice

## Output
The circuit produces an equalized output signal by combining controlled frequency bands.
